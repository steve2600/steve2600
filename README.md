## 🚀 Featured Projects

### ⚖️ [Verdicto-AI](https://github.com/steve2600/Verdicto-AI-1) — Legal Intelligence Platform &nbsp; `🥉 Hack Elite, National Level`
> LangChain · Weaviate · FastAPI · InLegalBERT · Docker · GitHub Actions

- Processes **1,000+ case documents (1950–2024)** with **80%+ verdict prediction accuracy**, cutting analysis time by **2×**
- Hybrid RAG pipeline with re-ranking → **35% better retrieval relevance**, P95 latency **<800ms**
- Fine-tuned InLegalBERT on **26,680 legal documents** → **+12% F1**, with automated bias detection across 7 categories (gender, caste, religion, etc.)

### 🏥 [T.R.I.A.G.E.](https://github.com/steve2600/T.R.I.A.G.E) — Clinical Risk Stratification Platform
> Python · scispaCy (BC5CDR) · medspaCy ConText · FastAPI

- Deterministic deterioration scoring over **11 physiological features** (SpO2, HR, BP, consciousness, etc.) using additive threshold rules
- Replaced regex extraction with **scispaCy NER + medspaCy negation resolution** — fixed a patient safety bug where negated symptoms were scored as active findings
- Surge forecasting via hour-of-day statistical averaging (μ + 1.5σ) — reproducible baseline over training-dependent models
- ML as an **optional enhancement layer with graceful degradation** — full triage works without the ML service
